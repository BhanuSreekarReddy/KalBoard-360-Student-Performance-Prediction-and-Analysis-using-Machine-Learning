Instructions to execute the code 

Included 3 files of R scripts to execute

1) Execution of files can be in any order but according to the results presented in the paper execute in the following order
   a) SVMLinear.r
   b) DecisionTree.r
   c) naiveBayes.r

2) For executing the individual scripts open them in RStudio.

3) Set the directory in "setwd" command to the path where the data file is stored in the system(Data File is attached in the submissions).

4) After setting the correct path, select all the commands and click run to get output.

5) Similarly, Execute the remaining r scripts.
